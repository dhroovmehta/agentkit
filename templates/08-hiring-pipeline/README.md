# Hiring Pipeline Agent

> Screen resumes, score candidates, schedule interviews, and draft communications.

## Quick Start

1. Copy `mcp-config.json` into Claude Desktop config
2. Create a Google Sheet for pipeline tracking
3. Import `workflow-n8n.json` into n8n
4. Add your job description to the screening prompt
5. Applications are automatically screened and scored

**Full setup guide:** See `setup-guide.md`

## What's Included

| File | Description |
|------|-------------|
| `system-prompt.txt` | 30-point scoring rubric, email templates, pipeline tracking |
| `mcp-config.json` | MCP config for email, calendar, drive, and file access |
| `workflow-n8n.json` | n8n workflow for resume screening pipeline |
| `setup-guide.md` | Setup guide |
| `README.md` | This file |

## Features

- 30-point candidate scoring rubric
- Automatic resume screening and ranking
- Interview invitation, rejection, and offer email templates
- Pipeline tracking with Google Sheets
- Weekly pipeline summary reports
