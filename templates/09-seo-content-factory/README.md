# SEO Content Factory

> Keyword research to published article in one automated pipeline.

## Quick Start

1. Copy `mcp-config.json` into Claude Desktop config
2. Import `workflow-n8n.json` into n8n
3. Send a keyword to the webhook
4. Receive a complete SEO-optimized article via email

**Full setup guide:** See `setup-guide.md`

## What's Included

| File | Description |
|------|-------------|
| `system-prompt.txt` | 5-stage SEO pipeline (research, analysis, outline, draft, checklist) |
| `mcp-config.json` | MCP config for web search, URL fetching, file access |
| `workflow-n8n.json` | n8n workflow for automated content production |
| `setup-guide.md` | Setup guide |
| `README.md` | This file |

## Pipeline Stages

1. Keyword Research & Clustering
2. Competitor Content Analysis (top 5 results)
3. SEO-Optimized Outline Generation
4. Full Article Draft (2,000+ words)
5. Pre-Publish SEO Checklist
