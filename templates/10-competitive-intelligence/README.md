# Competitive Intelligence Monitor

> Track competitor changes and get weekly alerts with strategic analysis.

## Quick Start

1. Copy `mcp-config.json` into Claude Desktop config
2. Create your competitor watch list (`competitors.json`)
3. Import `workflow-n8n.json` into n8n
4. Configure competitor names in workflow nodes
5. Activate — reports arrive every Monday

**Full setup guide:** See `setup-guide.md`

## What's Included

| File | Description |
