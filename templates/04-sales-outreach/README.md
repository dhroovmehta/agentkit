# Sales Outreach Automation

> Research leads, personalize emails, and run follow-up sequences automatically.

## Quick Start

1. Copy `mcp-config.json` into Claude Desktop config
2. Create a Google Sheet with your lead list
3. Import `workflow-n8n.json` into n8n
4. Configure your product description in the prompt
5. Test with one lead, then go live

**Full setup guide:** See `setup-guide.md`

## What's Included

| File | Description |
|------|-------------|
| `system-prompt.txt` | Research framework, email templates, compliance notes |
| `mcp-config.json` | MCP config for web search, email, and file access |
| `workflow-n8n.json` | n8n workflow for research + email + follow-up pipeline |
| `setup-guide.md` | Setup guide |
| `README.md` | This file |

## Features

- Automated lead research (company news, tech stack, pain points)
- Hyper-personalized first-touch emails
- 3-email follow-up sequence with varied angles
- CAN-SPAM and GDPR compliance guidelines
- Google Sheets CRM tracking
