# Weekly Business Intelligence Report

> Automated competitor and market intelligence delivered to your inbox every Monday.

## Quick Start

1. Copy `mcp-config.json` into Claude Desktop config
2. Import `workflow-n8n.json` into n8n
3. Configure search queries with your industry and competitors
4. Activate — reports arrive every Monday at 6 AM

**Full setup guide:** See `setup-guide.md`

## What's Included

| File | Description |
|------|-------------|
| `system-prompt.txt` | Report structure, analysis framework, data sources |
| `mcp-config.json` | MCP config for web search, URL fetching, file access |
| `workflow-n8n.json` | n8n workflow for automated weekly report generation |
| `setup-guide.md` | Setup guide |
| `README.md` | This file |

## Report Sections

1. Executive Summary
2. Competitor Updates (per competitor)
3. Industry News
4. Market Signals
5. Recommendations
6. Metrics Dashboard
