# Social Media Manager Agent

> Weekly content calendar, platform-optimized posts, and engagement monitoring.

## Quick Start

1. Copy `mcp-config.json` into Claude Desktop config
2. Import `workflow-n8n.json` into n8n
3. Configure your brand and industry in the workflow
4. Activate — content calendar generated every Sunday

**Full setup guide:** See `setup-guide.md`

## What's Included

| File | Description |
|------|-------------|
| `system-prompt.txt` | Content pillars, post templates, engagement framework |
| `mcp-config.json` | MCP config for research and file access |
| `workflow-n8n.json` | n8n workflow for automated calendar generation |

## Platforms Covered

- LinkedIn (5 posts/week)
- Twitter/X (5-7 posts/week + 1 thread)
- Instagram (2-3 posts/week)
