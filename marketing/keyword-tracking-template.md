## INTERPRETATION GUIDE

### What "Position" Means
- **Position 1-3:** "Sweet spot" — high CTR, most traffic
- **Position 4-10:** "Achievable" — shows up on page 1, get 30-40% of clicks
- **Position 11-20:** "Early stage" — page 2, low visibility, but still traffic
- **Position 21-100:** "Not ranking meaningfully" — not worth optimizing
- **No position:** "Not ranking" — keyword is new or needs more authority

### What "Impressions" Mean
- If a keyword has **100+ impressions/week** but position is 11-20, you're close to page 1. Add depth + internal links to push it up.
- If a keyword has **0 impressions** after 4+ weeks, it might not be targeting the right search intent. Check competitor content and adjust.

### What "CTR" (Click-Through Rate) Means
- If **CTR < 2%** but impressions are **100+**, your title/description isn't compelling. Rewrite it.
- If **CTR > 5%** on position 4-10, you have strong messaging — focus on getting to position 2-3 (bigger impact).

### Action Thresholds
**Trigger an optimization when:**
- A keyword has 100+ impressions but <2% CTR → Rewrite title/description
- A keyword is at position 8-12 with 50+ impressions → Add 300-500 words of content, add internal links
- A keyword has 0 impressions after 6+ weeks → Either remove it or check if page exists/ranks

## FAQ

**Q: "How do I see these keywords in GSC?"**
A: Go to Performance tab → scroll down to the keywords table. GSC only shows keywords you're actually ranking for. New keywords might not appear for 2-4 weeks.

**Q: "A keyword shows position 15 but I expected top 3. What went wrong?"**
A: Common reasons:
1. Competitor content is newer/better (update yours)
2. You're not the only one targeting it (increase backlinks)
3. Search intent mismatch (rewrite to match what users actually search for)
4. Page isn't getting enough internal links (add more from related content)

**Q: "Should I optimize for ALL 20 keywords?"**
A: No. Prioritize like this:
1. **Week 1-4:** Top 5 keywords (highest volume + achievable difficulty)
2. **Week 5-8:** Keywords 6-12
3. **Week 9-12:** Keywords 13-20 + long-tail variations
4. **Month 2+:** New keywords based on search trends

**Q: "Impressions are zero. Is the page indexed?"**
A: Check GSC → Coverage tab. If it says "Indexed" but 0 impressions after 4+ weeks, the content might not match search intent. Compare your page to competitors' top 3.
