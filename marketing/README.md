## Risk & Mitigation

### Risk: Slow Initial Traction
- **Why:** New sites take 4-6 weeks for Google to crawl and index
- **Mitigation:** Build backlinks aggressively in week 1-2 to signal importance to Google
- **Measurement:** After 4 weeks, should see 50-100 impressions in GSC

### Risk: Competitors Ranking Better
- **Why:** Established sites have more authority
- **Mitigation:** Focus on long-tail keywords (less competition), write deeper content (2,500+ words), acquire backlinks faster
- **Measurement:** Target top 3 positions for low-difficulty keywords first (<40 KD)

### Risk: Low Click-Through Rate
- **Why:** Compelling title/description not matching search intent
- **Mitigation:** A/B test title/description copy, monitor CTR in GSC, optimize for high-CTR benchmarks
- **Measurement:** CTR should be 2-5% for positions 4-10; if <2%, rewrite title/description

### Risk: Backlink Acquisition Stalls
- **Why:** Awesome lists reject PR, publications don't respond
- **Mitigation:** Diversify sources (don't rely on one strategy), follow up on pitches, add new tactics
- **Measurement:** Track weekly backlink count; if 0 new links for 2+ weeks, escalate and pivot

## Decision Framework

**Should I do [tactic]?**

Ask these 3 questions:

1. **Does it acquire backlinks OR drive targeted traffic?** (Yes/No)
2. **Can I execute it in <8 hours?** (Yes/No)
3. **Is it white-hat SEO?** (i.e., would I feel comfortable explaining it to Google directly) (Yes/No)

If all 3 are YES, do it. If any are NO, skip it.

## FAQ

**Q: How long before I see results?**
A: 4-6 weeks for initial indexing, 8-12 weeks for meaningful ranking movement. Patience required.

**Q: Should I rank for all 20 keywords?**
A: No. Prioritize top 5 (highest volume + achievable difficulty). Once those are in top 3, expand to 6-15.

**Q: What if backlinks aren't working?**
A: Check that they're high-quality (DR 50+, relevant anchor text, from trusted sites). Low-quality links don't move rankings.

**Q: Can I use paid ads to accelerate this?**
A: SEO is organic only. Paid ads (Google Ads, sponsored posts) are separate. Both should work together, but this guide is SEO-focused.

**Q: How do I know if I'm on pace?**
A: Monthly tracking. After 30 days, you should see 10-15 backlinks acquired. If slower, increase effort or shift tactics.
